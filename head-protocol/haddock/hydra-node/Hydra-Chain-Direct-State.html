data HeadStateKind #

Some Kind for witnessing Hydra state-machine's states at the type-level.

This is useful to

(a) Reads code evolving the state machine, as it makes transition more obvious from type-signatures; (b) Pattern-match on the HydraStateMachine without having to bother with non-reachable cases.
