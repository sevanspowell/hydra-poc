fromLedgerScript :: HasCallStack => Script era -> PlutusScript lang #

Convert a cardano-ledger's Script into a cardano-api's PlutusScript

Note that this function is unsafe in two manners:

(a) If the given script is a timelock script, it throws an impure exception; (b) If the given script is in a wrong language, it silently coerces it.
